## 🚀 Featured Project — PortfolioIntell

A production-deployed stock portfolio intelligence platform that combines backend engineering with real financial math.

- **Backend:** FastAPI · PostgreSQL · Redis · Docker · AWS EC2 + RDS
- **Auth:** JWT (OAuth2 password flow)
- **Algorithms:** Sharpe ratio, max drawdown, mean-variance portfolio optimization (Efficient Frontier)
- **Data:** yfinance · Alpha Vantage API
- **Infra:** GitHub Actions CI/CD · Nginx · Sentry error tracking · SlowAPI rate limiting
- **Live Demo:** http://52.3.232.204:3000
